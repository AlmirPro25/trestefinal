Como Funciona

O sistema de mídia integrado funciona da seguinte forma:

  1. O HTML contém placeholders no formato ai-researched-image://descrição e ai-researched-video://descrição
  2. O sistema busca imagens e vídeos no Pixabay com base nas descrições
  3. As URLs são comprimidas para ai-img://id e ai-vid://id e armazenadas no localStorage
  4. O componente HtmlPreview expande as URLs comprimidas para exibição
